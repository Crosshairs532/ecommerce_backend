###### after cloning the github repo . You have to install the dependencies. YOu may see below dependencies. 
```
    "cors": "^2.8.5",
    "dotenv": "^16.4.5",
    "express": "^4.19.2",
    "joi": "^17.13.1",
    "mongoose": "^8.4.0",
    "nodemon": "^3.1.0"
```
1. Install Cors:
...`npm i cors`
2. Install Dotenv
...`npm install dotenv --save`
3. Install express
...`npm i express`
